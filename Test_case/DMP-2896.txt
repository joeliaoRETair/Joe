__DMP-2896 Micro site Registration page Theme name display error
Summary:Check Micro site Registration page Theme name display

Step action:
1.進入Micro site Add page
2.Choose Page Type:Registration Menu
3.Drop Registraction Content 
4.檢查Registration Element Property 的 Theme名稱是否為:Register Page以及Register Page with Weixin Camera

預期結果:名稱正確
實際結果:與預期結果相同
